Hola Amigos!

Este servidor web, es una maquina virtual con sistema operativo Ubuntu y esta aprovisionada en el hypervisor "Hyper-V" de microsoft.

Para la adecuacion e instalacion de apache, se ejecutaron estos comandos;

Para generar las llaves SSH y permitir conexion sin clave: 


		ssh-keygen -t rsa -b 4096
		cat id_rsa
		cat id_rsa.pub
		vi authorized_keys
		chmod -R go= /home/apache/.ssh
		ls -la
		chmod 700 /home/apache/.ssh/
		ls -la
		chmod 644 /home/apache/.ssh/authorized_key
		ls -la
		sudo reboot now

Para configurar un IP fija en la tarjeta de red: 


		cd /etc/cloud/cloud.cfg.d
		vi subiquity-disable-cloudinit-networking.cfg
		cd /etc/netplan/
		vi 00-installer-config.yaml
		sudo vi 00-installer-config.yaml
		sudo netplan apply
		sudo netplan try
		ip a

Instalar Apache y cambiar el puerto: 


		sudo apt update
		sudo apt install apache2
		apache2 -version
		vi ports.conf
		sudo service apache2 restart
		service apache2 status

Activar Firewall y permitir puertos SSH y Apache 8088: 


		sudo ufw app list
		sudo ufw status
		sudo ufw allow 'Apache'
		vi /etc/default/ufw
		sudo vi /etc/default/ufw
		sudo ufw allow ssh
		sudo ufw enable
		sudo ufw allow 8088
		sudo ufw status
